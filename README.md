# Test

## Start server
```bash
python3 app.py
```

## API
### Get information about all products [get]:
```bash
<server_ip:port>/products
```

### Get information about product by product_id [get]:
```bash
<server_ip:port>/products/<product_id>
```

### Add new product [post]:
```bash
<server_ip:port>/products
```
